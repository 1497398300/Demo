# 像素机甲大战 - 游戏设计文档

## 游戏概览

**像素机甲大战 (Pixel Mecha Battle)** 是一款本地双人对战像素风格机甲格斗游戏，采用 HTML5 Canvas 实现，支持同键盘双人对战。

## 游戏机制

### 角色设定

双方属性**完全一致**，纯靠操作决胜负（仅外观颜色不同）。

| 属性 | 机甲·蓝 MECHA BLUE | 机甲·红 MECHA RED |
|------|-------------------|-------------------|
| 生命值 | 100 HP | 100 HP |
| 移动速度 | 3.0 | 3.0 |
| 近战伤害 | 15 | 15 |
| 远程伤害 | 10 | 10 |
| 大招伤害 | 25 | 25 |
| 能量恢复 | 0.4/帧 | 0.4/帧 |
| 外观 | 蓝色装甲+青色光效 | 红色装甲+橙色光效 |

### 操作方式

**P1 机甲·蓝:**
- WASD - 移动/跳跃
- J - 近战攻击（拳击）
- K - 远程攻击（能量弹）
- L - 大招（消耗 50 能量）
- I - 防御（减伤 75%，移速降低）

**P2 机甲·红:**
- 方向键 - 移动/跳跃
- 数字键 1 - 近战攻击
- 数字键 2 - 远程攻击
- 数字键 3 - 大招（消耗 50 能量）
- 数字键 0 - 防御

### 战斗系统

1. **近战攻击**: 短距离、低冷却，命中后增加连击数（额外伤害）
2. **远程攻击**: 发射直线飞行弹丸，中距离，中冷却
3. **大招**: 大型能量弹，伤害最高，长冷却，消耗 50 能量
4. **防御**: 按住减伤 75%，但移动速度降低 60%
5. **能量系统**: 自动恢复，满 100 时可释放大招
6. **击退效果**: 受击后产生击退和短暂硬直
7. **屏幕震动**: 大招命中和大伤害攻击触发屏幕震动

### 胜负判定

- 三局两胜制 (Best of 3)
- 每局限时 99 秒
- 生命值归零则对方获胜
- 时间耗尽时，生命值高者获胜
- 先获得 2 分者赢得比赛

## 代码架构

```
pixel-mecha-battle/
└── index.html          # 单文件游戏（HTML + CSS + JS）

核心模块:
├── Audio System        # Web Audio API 8-bit 音效
├── Particle System     # 粒子特效系统
├── Projectile System   # 弹丸系统
├── Mecha Class         # 机甲类（移动/攻击/防御/渲染）
├── Input Handler       # 键盘输入处理
├── Game Class          # 游戏主控（状态管理/循环/UI）
└── Rendering           # 双 Canvas 像素风格渲染
```

### 渲染方案

- **内部分辨率**: 640x360（16:9）
- **画面缩放**: 自适应窗口，整数倍缩放
- **像素风格**: `image-rendering: pixelated` + Math.floor 坐标
- **双 Canvas**: 离屏 Canvas 渲染 → 主 Canvas 缩放显示
- **帧率**: 60 FPS (requestAnimationFrame)

### 视觉设计

- 像素风机甲：程序化绘制（fillRect 拼接），非精灵图
- 背景：星空 + 城市剪影 + 动态灯光
- 特效：粒子系统（击中粒子、防御粒子、胜利粒子）
- UI：血条 + 能量条 + 计时器 + 比分指示器
- 标题画面：机甲展示 + 操作说明

## 技术要点

- 纯 HTML + CSS + JavaScript，无外部依赖
- Canvas 2D 渲染，imageSmoothingEnabled = false 实现像素风格
- Web Audio API 实现 8-bit 风格音效（方波/锯齿波/三角波）
- requestAnimationFrame 驱动游戏循环
- 碰撞检测：AABB 矩形碰撞
- 状态机管理游戏阶段（标题/对战/回合结束/游戏结束）
